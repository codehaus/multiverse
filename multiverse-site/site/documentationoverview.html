Recommended Papers

A lot of research has been done on transactional memory by many very smart people. Multiverse relies on a lot of the concepts and techniques discovered and invented by them. This is a listing of the most influencal papers for Multiverse.

Recommended Books

  • Transactional Information Systems: Theory, Algorithms, and the Practice of Concurrency Control and Recovery
  • Transactional Memory (Synthesis Lectures on Computer Architecture)
  • The Art of Multiprocessor Programming not only contains a lot of information about the theories behind concurrent programming, but also has a chapter about Software Transactional Memory
  • Concurrent Programming in Java(TM): Design Principles and Pattern (2nd Edition)
  • Java Concurrency in Practice the book about practical concurrent programming in Java.
  • Principles of Transaction Processing, Second Edition
  • One of the books of Thomas Kyte about Oracle. Oracle uses Multiversion Concurrency Control (MVCC), just like MySql + InnoDb and Postgresql. Multiverse essentially is a MVCC implementation:
    • Expert Oracle Database Architecture: Oracle Database Programming 9i, 10g, and 11g Techniques and Solutions, Second Edition
    • Expert One on One - Oracle
